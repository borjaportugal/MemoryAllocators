# MemoryAllocators
Implementation of multiple memory allocators, debug functionality and support for the stl containers.

## Implemented allocators
- GlobalAllocator<T>
- InlineAllocator<N, T>
- DebugInlineAllocator<N, T>
- FallbackAllocator<Primary, Fallback>
